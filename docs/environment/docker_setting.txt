Dockerインストール
*****************************

# Docker Desktop for Windowsインストール

# Docker Desktop起動
# Settings ⇒ Resources ⇒ WSL integration ⇒ Ubuntuを「ON」

# WSLログイン
$ wsl

# docker-composeファイルに移動

# イメージビルド＋コンテナ構築
$ docker compose up -d --build

*****************************
Dockerコマンド群
*****************************

# イメージビルド＋コンテナ構築(コード変更時)
$ docker compose up -d --build

# コンテナ構築
$ docker compose up -d

# コンテナ起動
$ docker compose start

# コンテナ停止
$ docker compose stop

# コンテナ削除
$ docker compose down

# コンテナログイン
$ docker compose exec app bash

# リアルタイムログ出力
$ docker compose logs -f app
$ docker compose logs -f app | grep "\[APACHE\]"
$ docker compose logs -f app | grep "\[SCHEDULER\]"
$ docker compose logs -f app | grep "\[WORKER\]"
